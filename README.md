# PreFlight-ML

PreFlight-ML is a pip-installable Python library for ML engineers and Kaggle-style researchers. It accepts a raw pandas DataFrame and a target column name, and returns a cleaned and feature-engineered DataFrame, a reusable serializable scikit-learn Pipeline, and a structured Report that logs every automated decision with its rationale.

## Installation

```bash
pip install preflight-ml
```
